# WizardKnight - 2D Platformer Game (Unity)
Game name: Woodland
Отправляйтесь в незабываемое приключение по зачарованному лесу в роли талантливой юной волшебницы! Сражайтесь со скелетами и озорными гоблинами, используя магические заклинания, и собирайте цветы.

🎮Управление:
* клавиатура
* геймпад

🎥 Видео геймплея: https://rutube.ru/video/75413a2b67b060ea9fd52d965cc610e2/

🛠️ Описание игрового процесса:
* Игрок перемещается с помощью бега и прыжков по платформам;
* Игрок может атаковать врагов, выпуская снаряды по ходу движения;
* Снаряды хранятся в пуле. При стрельбе снаряд берется из пула и контактирует с врагом или живет 3 сек и затем возвращается в пул;
* Существует по два вида врагов и наград. За создание вида отвечает фабрика, объекты хранятся в пуле. За отображение объектов отвечают спавнеры;
* Игрок может получить ранение за контакт с врагом, либо с платформой-шипы;
* В случае смерти игрок воскрешается у дерева. Враги восстанавливаются на сцене, а собранные награды - нет;
* Сохранение уровня происходит в меню паузы при выходе в главное меню;
* Враги содержат искусственный интеллект со следующими состояниями: покой, преследование, атака, смерть;
* Начиная с 3-его уровня, сцена содержит движущиеся платформы вверх-вниз и влево-вправо;
* На 4-ом уровне сцена темная, свет с бликами исходит от игрока;
* Игра содержит 1-ую сцену Bootstrap для запуска работы классов-менеджеров;
* Внедрение зависимостей осуществляется через VContainer.

* Canvas UI - отображение очков, жизни на игровой сцене, меню паузы, основное меню
