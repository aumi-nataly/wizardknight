# WizardKnight - 2D Platformer Game (Unity)
Отправляйтесь в незабываемое приключение по зачарованному лесу в роли талантливой юной волшебницы! Сражайтесь со скелетами и озорными гоблинами, используя магические заклинания, и собирайте цветы.

🎮Управление:
* клавиатура
* геймпад
